### QR Code Generator Note:
The current implementation uses a simplified QR generation algorithm for demonstration. For production use with guaranteed scanning compatibility, consider integrating a dedicated QR library like:
- `qrcode` (npm package)
- `qr-code-styling` (customizable QR codes)
- `qrcode.react` (React component)

All tools use modern JavaScript APIs:
- Canvas API (Image Resizer, QR Generator)
- Crypto API (secure random)
- Base64 encoding/decoding
- Regular Expressions
- Local Storage
